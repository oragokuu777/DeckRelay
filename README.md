# DeckRelay - Android to Mac Audio & Screen Relay

`DeckRelay` は、Android端末（スマートフォン、タブレット、プロジェクター等）のシステム音声を、同一Wi-Fi経由でMacへ高音質かつ低遅延に転送（リレー）するためのmacOS専用デスクトップアプリケーションです。

Androidデバイスの音声をMacへリレーして再生することをメイン目的としている、映像のリレー、テキスト共有（クリップボード連携）、バッテリー監視

---

## ✨ 主な機能 (Key Features)

- **🔊 音声リレー機能（メイン機能）**: Androidスマホ・タブレット・プロジェクター等のシステム音声を、同一Wi-Fi経由でMacへシームレスかつ高音質に転送します。
- **📱 画面・クリップボード連携（サブ機能）**: 音声転送に加え、画面表示やテキストのコピー＆ペースト共有にも対応。
- **🔋 バッテリー監視 ２％、８５%時にアラートを鳴らす。
- **⚡ 超軽量 & 省電力設計**: バックグラウンドでのCPU・メモリ消費を最小限に抑え、快適な動作環境を維持します。

---

## 🖥 動作環境 (System Requirements)

- **Mac側**: macOS 26 以降 (Apple Silicon 専用)
- **Android側**: Android 14 以降
- **ネットワーク**: 同一のWi-Fiネットワーク環境

---

## 🚀 使い方 (Quick Start)

### 1. アプリのセットアップ
1. 本リポジトリの`DeckRelay-1.0-arm64.zip` をダウンロードします。
2. Zipファイルを解凍し、`DeckRelay.app` を `アプリケーション` フォルダ（Applications）へ移動します。

### 2. 接続とリレーの開始
1. MacとAndroid端末（スマホ/タブレット/プロジェクター等）を同じWi-Fiネットワークに接続します。
2. `DeckRelay.app` を起動し、画面に表示される案内に従って接続を完了させます。
3. リレーを「ON」にすると、Androidの音声がMacから再生されます。
